# 🦊 راهنمای دانلود مدل‌های روباه

## 📁 تنظیم مسیر مدل‌ها

**مهم**: مدل‌ها در پوشه `models` داخل پروژه ذخیره می‌شوند (نه درایو C)

### 🚀 راه‌اندازی سریع:
```cmd
# 1. تنظیم مسیر مدل‌ها
setup_models_path.bat

# 2. دانلود مدل‌ها
download_models.bat
```

## 📋 مدل‌های مورد نیاز:

### 1. **llama4:scout** (مدل قدرتمند)
- **اندازه**: ~50GB
- **زمان دانلود**: 2-4 ساعت
- **کاربرد**: پیام‌های پیچیده و تحلیلی
- **ویژگی**: 109B parameters, MoE, Multimodal

### 2. **codellama:13b** (مدل کد)
- **اندازه**: ~7GB
- **زمان دانلود**: 30-60 دقیقه
- **کاربرد**: برنامه‌نویسی و کد
- **ویژگی**: تخصصی برای کد

### 3. **llama4:scout-q4** (مدل سریع)
- **اندازه**: ~10GB
- **زمان دانلود**: 45-90 دقیقه
- **کاربرد**: پیام‌های کوتاه و سریع
- **ویژگی**: کوانتیزه شده برای سرعت

### 4. **partai/dorna-llama3:8b** (فارسی - موجود)
- **اندازه**: ~8.5GB
- **وضعیت**: ✅ قبلاً نصب شده
- **کاربرد**: پیام‌های فارسی

## 🚀 دستورات دانلود:

### ویندوز:
```cmd
# تنظیم مسیر
setup_models_path.bat

# دانلود خودکار
download_models.bat

# یا دستی:
set OLLAMA_MODELS=%CD%\models
ollama pull llama4:scout
ollama pull codellama:13b
ollama pull llama4:scout-q4
```

### لینوکس/مک:
```bash
# تنظیم مسیر
export OLLAMA_MODELS="$(pwd)/models"

# دانلود
./download_models.sh
```

## 📊 مجموع فضای مورد نیاز:
- **کل**: ~67 گیگابایت
- **محل ذخیره**: `D:\fox\models` (داخل پروژه)
- **زمان کل**: 3-6 ساعت (بسته به سرعت اینترنت)

## ⚡ دانلود سریع (فقط ضروری):
اگر فضای کم داری، فقط این دو رو دانلود کن:
```bash
set OLLAMA_MODELS=%CD%\models
ollama pull codellama:13b      # برای کد (7GB)
ollama pull llama4:scout-q4    # برای سرعت (10GB)
```

## 🔍 بررسی مدل‌های نصب شده:
```bash
set OLLAMA_MODELS=%CD%\models
ollama list
```

## 🔧 تنظیم دائمی متغیر محیطی:

### روش 1: PowerShell (Administrator):
```powershell
[Environment]::SetEnvironmentVariable("OLLAMA_MODELS", "D:\fox\models", "User")
```

### روش 2: GUI:
1. کلیک راست روی "This PC"
2. Properties → Advanced System Settings
3. Environment Variables
4. اضافه کردن متغیر جدید:
   - نام: `OLLAMA_MODELS`
   - مقدار: `D:\fox\models`
